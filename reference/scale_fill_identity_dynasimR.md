# Fill scale for combatant identity

As
[`scale_color_identity_dynasimR()`](https://rabanheller.github.io/dynasimR/reference/scale_color_identity_dynasimR.md)
but for the `fill` aesthetic.

## Usage

``` r
scale_fill_identity_dynasimR(...)
```

## Arguments

- ...:

  Arguments passed to
  [`ggplot2::scale_fill_manual()`](https://ggplot2.tidyverse.org/reference/scale_manual.html).

## Value

A ggplot2 scale object.
