# Autonomy-level trade-off plot: KIA reduction vs. IHL compliance

Autonomy-level trade-off plot: KIA reduction vs. IHL compliance

## Usage

``` r
plot_al_tradeoff(al_result, label_points = TRUE, highlight_optimal = TRUE)
```

## Arguments

- al_result:

  A `dynasimR_al_analysis` object.

- label_points:

  Logical. Annotate points with AL labels. Default `TRUE`.

- highlight_optimal:

  Logical. Circle the optimal AL. Default `TRUE`.

## Value

A ggplot2 object.
